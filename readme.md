# Parallax Error in Commercial Wearable Eye Trackers: Effects of Viewing Distance on Gaze Accuracy
This repository contains the materials, analysis setup and code, and parallax model for the paper Niehorster, D. C., Pettenella, A., Marano, G., Carminati, M., Merigo, L. & Nyström, M. (submitted). Parallax Error in Commercial Wearable Eye Trackers: Effects of Viewing Distance on Gaze Accuracy.

When using the contents of this repository in your work, please cite Niehorster et al. (submitted).

## Contents
- `analysis`: Code for further analysis of data after processing with gazeMapper
- `gazeMapper_project`: Preconfigured gazeMapper project used for the analysis of recordings made with the provided materials
- `poster_walking`: A0 poster used for the walking task
- `stimulus`: Code implementing the procedure for the experiment, which includes presenting the visual stimuli for validation and the sitting task
- `parallax_model.m`: MATLAB code implementing the parallax model. Generates Figures 1B and 1C in the paper.
